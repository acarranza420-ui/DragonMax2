# DragonMax RD Ultimate v6.0

Legal Kodi repository and wizard starter for a DragonMax fantasy/anime themed build.

## Install in Kodi
1. Upload everything inside `dist` to the root of your GitHub Pages repo.
2. Add your GitHub Pages URL in Kodi File Manager.
3. Install `repository.dragonmax-1.6.0.zip`.
4. Install DragonMax RD Wizard from the repository.

## Included
- Repository ZIP
- Wizard ZIP
- addons.xml + MD5
- v6 docs and build manifest

## Not Included
- Piracy add-ons
- Unauthorized streaming sources
- Passwords or tokens
- Copyrighted character assets
- Commercial fonts
