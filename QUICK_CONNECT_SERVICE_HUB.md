# Quick Connect Service Hub

A single first-launch panel for service sign-ins.

## Services
- Real-Debrid cloud / authorized files
- Trakt
- Plex
- Jellyfin
- Emby
- YouTube
- OpenSubtitles
- Weather provider

## Flow
1. Select service.
2. Show QR code or device login URL when supported.
3. User authorizes from phone/computer.
4. Status changes to Connected.

Never store raw passwords or private tokens in a public repo.
