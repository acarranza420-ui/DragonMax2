# Dragon Portal 2.0

Phone dashboard concept launched by QR code.

## Functions
- Upload wallpapers
- Upload videos
- Change theme
- Change voice
- View storage
- Schedule maintenance
- Manage profile preferences
- Restart Kodi manually if supported by your environment

## Hosting Options
- Local web server on your network
- NAS/SMB folder
- Cloud upload page
- GitHub Pages for static content only
